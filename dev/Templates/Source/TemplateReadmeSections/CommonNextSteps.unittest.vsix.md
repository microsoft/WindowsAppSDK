## Common next steps

- **Add a test class**: *Add → Class* — any `.cs` file with
  `[TestClass]` and `[TestMethod]` attributes is picked up by Test
  Explorer.
- **Reference the code under test**: *Add → Project Reference* and tick
  the project being tested, or *Manage NuGet Packages* to pull in a
  library.
- **Update the SDK**: open *Manage NuGet Packages*, switch to the
  *Updates* tab, and update **Microsoft.WindowsAppSDK**.
