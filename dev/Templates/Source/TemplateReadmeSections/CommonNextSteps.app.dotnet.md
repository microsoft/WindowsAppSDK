## Common next steps

- **Add items**: `dotnet new winui-page`, `winui-window`, `winui-usercontrol`, `winui-templatedcontrol`, `winui-dialog`, `winui-resourcedictionary`, `winui-resw`. Run `dotnet new list winui` for the full list.
- **Add a unit test project**: `dotnet new winui-unittest -n MyApp.Tests`.
- **Update the SDK**: `dotnet add package Microsoft.WindowsAppSDK`.

For more, see [WinUI 3 Documentation](https://learn.microsoft.com/windows/apps/winui/winui3/).
