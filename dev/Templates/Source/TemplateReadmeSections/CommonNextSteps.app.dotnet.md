## Common next steps

- **Add an item**: `dotnet new winui-page`, `winui-window`,
  `winui-usercontrol`, `winui-templatedcontrol`, `winui-dialog`,
  `winui-resourcedictionary`, or `winui-resw`. `dotnet new list winui`
  shows every available template.
- **Add a unit test project**: `dotnet new winui-unittest -n MyApp.Tests`
  (pre-wired for MSTest with MSIX packaging).
- **Update the SDK**: `dotnet add package Microsoft.WindowsAppSDK`.
