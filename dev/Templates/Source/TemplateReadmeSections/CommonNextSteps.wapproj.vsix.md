## Common next steps

- **Edit the manifest**: open `Package.appxmanifest` in the manifest designer or as XML (capabilities, display name, logos, file associations, protocols).
- **Add a project to the package**: right-click `.wapproj` → *Add → Reference*.
- **Add WinUI items**: *Add → New Item → WinUI* on the **app project** (not the `.wapproj`) for pages, windows, controls, dialogs, resource dictionaries, `.resw`.
- **Update the SDK**: *Manage NuGet Packages* on the app project → *Updates → Microsoft.WindowsAppSDK*.

For more, see [Package and deploy a WinUI 3 desktop app](https://learn.microsoft.com/windows/apps/package-and-deploy/).
