## Build and run

- Press **F5** in Visual Studio to build, deploy, and launch the app.
- For command-line builds, use:

```powershell
msbuild /restore /p:Configuration=Debug /p:Platform=x64
```

For advanced MSBuild packaging options, see [MSBuild properties for packaging projects](https://learn.microsoft.com/windows/msix/desktop/desktop-to-uwp-packaging-dot-net#build-the-package-from-the-command-line).
