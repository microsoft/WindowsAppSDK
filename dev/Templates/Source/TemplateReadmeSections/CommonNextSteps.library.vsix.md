## Common next steps

- **Add items**: *Add → New Item → WinUI* (user controls, templated controls, resource dictionaries, `.resw`).
- **Consume the library**: from a WinUI 3 desktop app project in the same solution, *Add → Project Reference* and tick this library.
- **Update the SDK**: *Manage NuGet Packages → Updates → Microsoft.WindowsAppSDK*.

For more, see [WinUI 3 Documentation](https://learn.microsoft.com/windows/apps/winui/winui3/).
