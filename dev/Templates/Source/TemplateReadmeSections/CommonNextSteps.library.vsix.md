## Common next steps

- **Add an item**: *Add → New Item → WinUI* offers user controls,
  templated controls, resource dictionaries, and `.resw` files
  suitable for a class library.
- **Consume the library**: from a WinUI 3 desktop app project in the
  same solution, *Add → Project Reference* and tick this library.
- **Update the SDK**: open *Manage NuGet Packages*, switch to the
  *Updates* tab, and update **Microsoft.WindowsAppSDK** to the latest
  version.
