## Prerequisites

- Windows 10 version 1809 (build 17763) or newer.
- Enable **Developer Mode**: *Settings → System → For developers*.
- Install the .NET SDK matching `<TargetFramework>` in the `.csproj`.

For detailed setup, see [Windows App SDK Prerequisites](https://learn.microsoft.com/windows/apps/windows-app-sdk/set-up-your-development-environment).
