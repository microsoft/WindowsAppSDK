## Common next steps

- **Add items**: `dotnet new winui-usercontrol`, `winui-templatedcontrol`, `winui-resourcedictionary`, `winui-resw`. Run `dotnet new list winui` for the full list.
- **Consume the library**: `dotnet add reference ../path/to/this.csproj`, or `dotnet pack` and reference the produced `.nupkg`.
- **Update the SDK**: `dotnet add package Microsoft.WindowsAppSDK`.

For more, see [WinUI 3 Documentation](https://learn.microsoft.com/windows/apps/winui/winui3/).
