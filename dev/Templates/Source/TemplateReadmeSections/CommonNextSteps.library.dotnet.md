## Common next steps

- Add items: `dotnet new winui-usercontrol`, `winui-templatedcontrol`, `winui-resourcedictionary`, `winui-resw` (full list: `dotnet new list winui`).
- Consume: `dotnet add reference ../path/to/this.csproj`, or `dotnet pack` and reference the produced `.nupkg`.
- Update SDK: `dotnet add package Microsoft.WindowsAppSDK`.

See [WinUI 3 docs](https://learn.microsoft.com/windows/apps/winui/winui3/).
